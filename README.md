# hanbit-b2b-deploy

Hanbit B2B promotion landing page deployed on Vercel.

## Live URL

https://hanbit-b2b-deploy.vercel.app/

## Project Structure

- `index.html` - main landing page
- `courses-data.json` - course data used by the page
- `promo-config.json` - promotion/configuration data
- `api/lead.js` - lead submission API
- `api/leads.js` - lead guide/status API

## Deployment

This project is intended to be deployed with Vercel.

The local `.vercel` directory and generated backup HTML files are intentionally excluded from Git.
